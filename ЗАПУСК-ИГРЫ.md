# Проект Горизонт — Инструкция по запуску

## Подготовка (один раз)

### 1. Установка зависимостей

```bash
cd /Users/aleksandrkalentev/IdeaProjects/newgorizont
npm run install:all
```

### 2. Настройка туннеля (выберите один вариант)

#### Вариант А: Cloudflare Tunnel (без регистрации)

```bash
# Установка
brew install cloudflared

# Настройка DNS (обязательно!)
# Системные настройки → Wi-Fi → Детали → DNS → добавить:
# 1.1.1.1
# 8.8.8.8

# Или через терминал:
networksetup -setdnsservers Wi-Fi 1.1.1.1 8.8.8.8
```

#### Вариант Б: ngrok (нужна бесплатная регистрация)

```bash
# Установка
brew install ngrok

# Регистрация (один раз)
# 1. Откройте https://ngrok.com/signup
# 2. Создайте бесплатный аккаунт
# 3. Скопируйте токен: https://dashboard.ngrok.com/get-started/your-authtoken
# 4. Выполните:
ngrok config add-authtoken ВАШ_ТОКЕН
```

## Запуск игры

### Автоматический (рекомендуется)

```bash
./start-game.sh
```

### Ручной (два терминала)

**Терминал 1 — Сервер:**
```bash
npm run prod
```

**Терминал 2 — Туннель:**
```bash
# Cloudflare:
cloudflared tunnel --url http://localhost:3000

# Или ngrok:
ngrok http 3000
```

## После запуска

В консоли появится публичный URL:
```
🌍 Публичный URL:
   https://random-words.trycloudflare.com
```

### Порядок действий:

1. **Откройте админку** через публичный URL + `/admin`
2. **Введите пароль:** `horizon2024`
3. Перейдите в **«Участники»** → нажмите **«📱 QR-коды»**
4. **Распечатайте** QR-коды или покажите на экране
5. Игроки сканируют свой QR-код телефоном

## Решение проблем

### DNS ошибки в Cloudflare Tunnel
```
error looking up Cloudflare edge IPs
```
**Решение:** Добавьте DNS серверы 1.1.1.1 и 8.8.8.8

### ngrok требует authtoken
```
authentication failed: Usage of ngrok requires authtoken
```
**Решение:** Зарегистрируйтесь на ngrok.com и добавьте токен

### Порт 3000 занят
```bash
pkill -f "node server/dist"
```

### QR-коды не работают
Убедитесь что открыли админку через **публичный URL туннеля**, а не localhost
