# Horizon Map — Интерактивная игра

Многопользовательская интерактивная игра для совместного построения города будущего.

## Структура проекта

```
horizon-map/
├── server/          # Backend: Node.js + Express + Socket.IO
├── client/          # Frontend: React 18 + Vite + Tailwind
├── package.json     # Корневые скрипты для запуска
└── .env.example     # Пример переменных окружения
```

## Требования

- Node.js 18+
- npm 9+

## Установка

```bash
# Клонируйте репозиторий
git clone <repo-url>
cd horizon-map

# Установите все зависимости
npm run install:all

# Скопируйте файл окружения
cp .env.example .env
```

## Запуск в режиме разработки

```bash
npm run dev
```

Это запустит:
- **Сервер** на порту 3000
- **Клиент** на порту 5173 (с прокси на сервер)

## Маршруты приложения

| Маршрут | Описание |
|---------|----------|
| `/map` | Проекция карты (для большого экрана) |
| `/admin` | Панель управления ведущего |
| `/play/:token` | Мобильный интерфейс игрока |

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `ADMIN_PASSWORD` | Пароль для админ-панели | `horizon2024` |
| `SESSION_SECRET` | Секретный ключ сессий | — |
| `PORT` | Порт сервера | `3000` |

## Сборка для production

```bash
# Собрать клиент и сервер
npm run build

# Запустить production сервер
npm start
```

## Архитектура

### Сервер
- Express.js для HTTP-маршрутов
- Socket.IO для real-time коммуникации
- SQLite для хранения состояния игры
- TypeScript для типобезопасности

### Клиент
- React 18 с hooks
- Vite для быстрой разработки
- Tailwind CSS для стилизации
- Socket.IO Client для real-time обновлений

## Игровые зоны

- **Центр** — административный центр города
- **Жилая зона** — жилые кварталы
- **Промышленная зона** — заводы и производство
- **Зелёная зона** — парки и экология
- **Неизвестная зона** — скрытая территория
