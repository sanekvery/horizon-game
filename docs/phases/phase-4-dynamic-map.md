## Socket-события

| Событие | Данные | Описание |
|---------|--------|----------|
| `map:resource-contributed` | `{ zone, resource, amount, roleId, roleName, newTotal }` | Ресурсы внесены в зону |
| `map:zone-upgraded` | `{ zone, fromLevel, toLevel }` | Зона улучшена |
| `map:event-started` | `{ eventId, isDilemma? }` | Событие активировано |
| `map:event-ended` | `{ eventId }` | Событие завершено |

---

## Как проверить

1. Открыть `/map?session=CODE`
2. В админке дать игроку ресурсы
3. Игрок вносит ресурсы в зону → видна анимация потока и "+N"
4. Индикаторы ресурсов отображаются на каждой зоне
5. Админ улучшает зону → красивая анимация Level Up
6. Активировать событие → пульсирующий оверлей
7. `/map?session=CODE&projector=true` → режим проектора (без индикаторов)
