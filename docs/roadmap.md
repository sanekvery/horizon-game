
### Пользователи системы

| Роль | Описание | Авторизация | Статус |
|------|----------|-------------|--------|
| **Гость** | Может играть по QR/ссылке | Нет | ✅ Работает |
| **Фасилитатор** | Создаёт и ведёт игры | Логин + пароль | ✅ Реализовано |
| **Админ** | Управляет фасилитаторами, видит всё | Логин + пароль | 📋 Планируется |

## 4. Инфраструктура

### Деплой
После каждой фазы:
```bash
git pull origin master
docker-compose down
docker-compose up -d --build
docker-compose logs -f
```

### Переменные окружения
```
ADMIN_PASSWORD=...        # Пароль для legacy доступа к админке
DB_PASSWORD=...           # Пароль PostgreSQL
JWT_SECRET=...            # Секрет для JWT токенов
```
