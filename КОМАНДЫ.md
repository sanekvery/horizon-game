# Проект Горизонт — Команды

## Быстрый запуск

```bash
cd /Users/aleksandrkalentev/IdeaProjects/newgorizont
./start-game.sh
```

По умолчанию использует ngrok (уже настроен).

---

## Варианты запуска

| Команда | Описание |
|---------|----------|
| `./start-game.sh` | ngrok (по умолчанию) |
| `./start-game.sh --ngrok` | ngrok явно |
| `./start-game.sh --cloudflare` | Cloudflare Tunnel |
| `./start-game.sh --local` | Локально (все в одной WiFi) |

---

## Ручной запуск (два терминала)

**Терминал 1 — Сервер:**
```bash
cd /Users/aleksandrkalentev/IdeaProjects/newgorizont
npm run prod
```

**Терминал 2 — Туннель:**
```bash
ngrok http 3000
```

## После запуска

В консоли появится URL, например:
```
🌍 Публичный URL:
   https://xxx-xxx.ngrok-free.app
```

| Страница | URL |
|----------|-----|
| **Админка** | `https://xxx.ngrok-free.app/admin` |
| **Карта** (проектор) | `https://xxx.ngrok-free.app/map` |
| **Игрок** | QR-код или `/play/TOKEN` |

**Пароль админки:** `horizon2024`

---

## QR-коды для игроков

1. Открой админку через публичный URL
2. Войди с паролем `horizon2024`
3. Перейди в **«Участники»**
4. Нажми **«📱 QR-коды»**
5. Распечатай или покажи на экране

---

## ⚠️ Важно для игроков (ngrok)

При первом открытии ссылки через ngrok появится **страница предупреждения**:

> "You are about to visit..."

Игроки должны нажать кнопку **"Visit Site"**, чтобы попасть в игру.

**Совет:** Попросите одного игрока открыть ссылку первым и показать остальным, что нужно нажать.

